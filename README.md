# Source-Engine-On-Android-Helper
Helper to download the Source Engine port for Android and the Games.
# To install
Download Termux from F-Droid. run 
```bash
pkg update
```
and run the below
```bash
bash <(curl -sL https://raw.githubusercontent.com/HackSucks/Source-Engine-On-Android-Helper/main/setup.sh)
```
then just run
```bash
seoah
```
and you are good!

# Notes
Install directories have /sdcard prefix. which is emulated/0 in termux. so dont type /sdcard/game, only type game.
